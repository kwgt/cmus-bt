# cmus-bt
Bluetooth AVRCP wrapper for [cmus](https://cmus.github.io/).

## Usage
```
cmus-bt [options] {NAME|DEVICE-ADDRESS}

options:
    -l, --list-available-device
        --log-file=FILE
        --log-level=FLEVEL
```

### options
<dl>
  <dt>-l, --list-available-device</dt>
  <dd>show available device list</dd>
</dl>
